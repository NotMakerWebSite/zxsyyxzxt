### 三、功能描述
系统分为（身份）：用户、管理员。

**用户功能：**
1. 登录
2. 注册
3. 浏览自习室、详情。
4. 评论区。
5. 自习室选座预约。
6. 个人信息。
7. 我的订单。
8. 我的收藏。


**管理员功能：**
1. 登录
2. 统计分析。
3. 用户管理。
4. 公告管理。
5. 自习楼管理。
6. 自习室管理。
7. 座位管理。
8. 订单管理。
9. 评价管理。
10. 管理员管理。
